## ðŸš€ Installation Methods

### Method 1: Outlook Desktop (Windows)

1. Open `generate-signature.html` in browser
2. Click "Copy Signature"
3. In Outlook: **File > Options > Mail > Signatures**
4. Click "New", name it
5. Paste in the editor (Ctrl+V)
6. Set as default
7. Done!

### Method 2: Outlook Desktop (Mac)

1. Open `generate-signature.html` in browser
2. Click "Copy Signature"
3. In Outlook: **Preferences > Signatures**
4. Click "+" to create new
5. Paste in editor (Cmd+V)
6. Close preferences (auto-saves)
7. Done!

### Method 3: Outlook Web (Office 365)

1. Open `generate-signature.html` in browser
2. Click "Copy Signature"
3. Go to outlook.office.com
4. Settings âš™ï¸ > View all settings
5. **Mail > Compose and reply > Email signature**
6. Paste in editor
7. Save
8. Done!
