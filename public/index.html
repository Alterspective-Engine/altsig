üìñ How to Install Your Signature in Outlook

For New Outlook (Windows):
  1. Click the Settings gear icon (‚öôÔ∏è) in the top right
  2. Navigate to Accounts > Signatures
  3. Select the email account you want to add the signature to
  4. Click New signature
  5. Give your signature a name (e.g., "Professional Signature")
  6. Click in the signature editor box and paste your copied signature (Ctrl + V)
  7. Click Save
  8. Optional: Under "Default signatures", choose when to use this signature:
    • New messages - automatically in new emails
    • Replies/forwards - automatically in replies
  9. Click Save again
For Classic Outlook (Windows):
  1. Open Outlook and click New Email
  2. In the message window, go to the Message tab
  3. Click Signature > Signatures...
  4. Click the New button
  5. Type a name for your signature (e.g., "Professional Signature")
  6. Click OK
  7. In the "Edit signature" box, paste your copied signature (Ctrl + V)
  8. Optional: Under "Choose default signature":
    • Select your email account
    • Choose the signature for "New messages"
    • Choose the signature for "Replies/forwards"
  9. Click OK to save
üìå Note: If you use both Outlook Web and Outlook Desktop, you'll need to set up signatures separately for each. If you have Outlook Desktop on multiple computers, each will need its own setup.
For Outlook for Mac:
  1. On the top menu bar, click Outlook > Settings
  2. Under "Email", select Signatures
  3. Click the + (plus) button to add a new signature
  4. Type a name for your signature (e.g., "Professional Signature")
  5. In the signature editor, paste your copied signature (Command + V)
  6. Format the signature if needed using the toolbar options
  7. Click Save
  8. Close the signature editor window
  9. Optional: To set as default:
    • In the Signatures window, select your account
    • Choose the signature for "New messages"
    • Choose the signature for "Replies/forwards"
  10. Close the Signatures window
üí° Quick Tip: To manually add your signature to an email, click in the message body, then on the Ribbon select Signature and choose your signature from the list.
üìå Note: If you use Outlook on multiple devices, you'll need to add the signature to each device separately.
